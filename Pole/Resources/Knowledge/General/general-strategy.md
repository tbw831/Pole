# 赛车策略 — 进站窗口 (Pit Window)

进站窗口是车队预判最佳进站时机的圈数区间。
影响因素:
- 轮胎衰减曲线(软胎掉速快,硬胎掉速慢)
- 直道速度差(对手在哪条胎,会不会 undercut 我)
- 安全车 / VSC 概率(可大幅缩短进站损失,甚至变成"免费换胎")
- 队友策略对冲(有人一停 + 队友两停)
- 起跑顺序(后排车手通常更早进站清空交通)

经典策略术语:
- **Undercut**:比前车早进站,新胎跑几圈赶位置后追平 + 反超
- **Overcut**:比前车晚进站,旧胎多跑几圈,等对方 out-lap 慢时追上
- **Stay out**:不进站赌安全车 / 雨势变化

# 赛车策略 — 拖车 (Slipstream / Tow)

直道高速时,前车后面的"低气压袋"减少阻力 → 后车顶速更高 = 拖车效果。
F1 / MotoGP / FE 用法:
- 排位赛:跟在快车后跑出更快单圈(双车策略)
- 正赛超车:DRS 区前先 tow 一段拉开速度差,出弯刹车点反超
- 长直道末端 slipstream + late braking 是典型超车套路

注意:跟太近 1 秒内会受前车 dirty air 影响下压力,弯角失利。
所以 tow 通常用在直道,不用在 hairpin 后接长弯。

# 赛车策略 — 安全车 (Safety Car) 利用

SC 出场让后车追上前车,甩开的差距瞬间作废 — 双刃剑。
利:
- 进站损失 ↓ 50% — 因为别人也减速
- 重启时新胎对比旧胎优势更大
弊:
- 已经 build up 的领先优势全没
- 重启乱战容易撞车出局

策略组通常:
- 领先且距离近 → 不进站,押对手不进
- 领先且差距大 → 趁机进站锁胜利
- 中段后段 → 几乎一定进站 free pit stop

# 赛车策略 — 起步 (Race Start)

F1 起步:绿灯亮 + 油门 + 离合 = 加速,前 100 m 是关键。
- T1 内线优势:能争 line 的就争,但避免相撞
- 不要在 T1 急刹后压低 line 否则被夹击
- 起步前的 "anti-stall" 系统会自救小失误,但失败就熄火

MotoGP 起步:红灯灭起步,前 50 m 通常分胜负。
- 抬头 (wheelie) 控制重要 — 起步时前轮抬太高跟不动
- Holeshot device(前叉锁住降低重心)是关键工具

发车顺序高的车更难争 — 后排发车 + 雨战 + 安全车 = 上车好戏经典剧本。

# 赛车策略 — 轮胎管理 (Tire Management)

干胎三档:软 (Soft, 最快但只能跑 ~20 圈)/ 中 (Medium) / 硬 (Hard)。
软胎 = 最快单圈但热衰快 + 颗粒化(graining)+ 滑行(blistering)。
硬胎 = 启动温度难达成 + 但稳定后能跑 50+ 圈。
车手 management:
- 弯心慢一点出弯快一点 → 减温度
- 不要 lock-up(锁死胎面)→ 局部磨损直接报废轮胎

雨胎(intermediates 半雨 / wets 全雨):
- inter:水量小 → 中,水量大就摔
- wets:大水中开槽花纹,但干面跑 5 圈热衰报废

# 赛车策略 — 排位赛 (Qualifying)

F1 Q1/Q2/Q3 三轮:
- Q1:基本所有车都跑 1-2 套软胎,只看赛季中下游谁掉队
- Q2:Top 10 战开始,top teams 选择是否冲圈或保胎给正赛
- Q3:全力冲 1-2 圈,12 分钟内最快单圈定 pole
策略组件:tire warm-up,track evolution(后期跑的圈快是因为赛道吸胎屑后抓地力高),
traffic management(避免遇慢车)。

MotoGP Q2:
- 12 分钟内 2-3 个出门冲圈,中间回 box 换软胎
- 赛车 setup 已经在 FP3 调好,Q2 只看车手

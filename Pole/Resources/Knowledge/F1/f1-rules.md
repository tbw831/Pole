# F1 周末赛制

F1 标准赛周末三天:周五两节练习赛 (FP1 / FP2),周六排位赛之前一节练习 (FP3),
周六下午排位赛 (Q1 / Q2 / Q3 三轮淘汰),周日正赛 (Race / Grand Prix)。
排位赛 Q1 淘汰最慢 5 名,Q2 再淘汰 5 名,Q3 由 10 人争最后正赛发车顺序。
车手在 Q3 的最快单圈即为正赛 pole position 起跑顺序。

# F1 冲刺赛周末 (Sprint Weekend)

部分大奖赛是 Sprint 格式,2024 起标准为:周五 FP1 + Sprint Qualifying (SQ),
周六上午 Sprint (短赛,~100km, 计入车手榜分数前 8 名 8/7/6/5/4/3/2/1 分),
周六下午正赛排位赛,周日正赛照常。Sprint 格式让周五就有竞技内容。

# F1 2026 规则大改概览

2026 是 F1 史上最大规则变革年之一,跟 2014 hybrid + 2017 wide track + 2022 ground effect 一档。
主要改动:
1. **动力总成**:V6 1.6L turbo hybrid 保留,但电动占比从 ~20% 升到 ~50%(电池输出大幅提升)
2. **燃料**:强制 100% 可持续 (sustainable) 燃料,FIA 测试合规
3. **车体尺寸**:更轻(整车 -30kg 左右),更短更窄,贴地高度规则也改
4. **空气动力学**:**取消 DRS**,改为 Active Aerodynamics(主动空气) + Manual Override Mode
5. **MGU-H 取消**:简化动力总成,降准入门槛(吸引 Audi / 福特等新厂)
6. **预算帽**:维持但额度调整,运营成本控制延续
背景:平衡"性能 + 可持续 + 新厂入场"三要素,FIA 2022 起持续推动。

# F1 主动空气动力学 (Active Aerodynamics, 2026 起)

2026 新规彻底取代了过去 15 年(2011-2025)的 DRS 系统。
车手在直道上可以**主动调节前翼 + 后翼**两个翼面,不再依赖"跟车 1 秒"才能开。
- 直道:前后翼降到低阻力档(low-drag mode),顶速更高
- 弯前:翼面回到高下压力档(high-downforce mode),保过弯抓地

跟旧 DRS 区别:
1. 不限定 DRS 区 — 任何直道都能切到 low-drag
2. 不需要跟前车 1 秒触发 — 全场全车都能用
3. 切换可由车手手动 / 系统自动配合(各队策略不同)
4. 排位赛 / 正赛规则一致

# F1 Manual Override Mode (2026 超车助力)

为补偿"DRS 取消后超车难"问题,2026 引入 Manual Override Mode (MOM)。
- 仅"追击者"可用(跟前车 1 秒内,类似旧 DRS 的"激活权")
- 触发后给电池更高输出上限(瞬时 push 功率提升)
- 持续时间和次数受 race 配额限制(每场可激活 N 次)
- 红黄旗 / SC / VSC / 雨条件下禁用(跟旧 DRS 一致)
设计思路:把"主动空气"给所有车均等享受 → 平衡赛车;把"超车助力"给后车独占 → 保留竞技性。

# F1 DRS (历史规则,2011-2025)

⚠️ **DRS 在 2026 已废除**,见上面 Active Aero / MOM 条目。本段仅作历史背景。
DRS = Drag Reduction System,可调后翼。启用条件:
1. 必须在 FIA 划定的 DRS 区(每条赛道 1-3 段直道);
2. 跟前车距离 ≤ 1 秒(detection point 检测);
3. 比赛第 3 圈起才允许;
4. 红黄旗 / 安全车 / VSC / 雨胎条件禁用。
排位赛中 DRS 全程开放,不需要跟车触发。
2011 引入用以解决加 KERS 后超车仍难的问题,运行 15 个赛季,2026 被 Active Aero 取代。

# F1 积分制度

正赛前 10 名得分:25/18/15/12/10/8/6/4/2/1。
最快圈 (FL) +1 分,前提是车手在前 10 名内完赛(2024 起规则收紧)。
Sprint 前 8 名:8/7/6/5/4/3/2/1 分。
赛季结束积分最高的车手 = 世界冠军,车队总分最高的 = 车队冠军。

# F1 进站规则

每场必须用至少 2 种不同配方干胎(雨胎不算);全雨场则免除该约束。
进站速度限制 60 km/h (部分赛道 80 km/h),超速罚 5-10 秒赛后追加。
强制进站窗口:DRS 区开启后任何时间都可进站,但策略组通常在 18-30 圈窗口操作。
轮胎牌号:C1(最硬)→ C2 → C3 → C4 → C5(最软),每周末选 3 种供给。
2024 起 Pirelli 也提供 C0(超硬,极少赛道用)。

# F1 罚时规则

常见罚则:
- 5 秒罚时(Stop&Go 5s):下次进站前停 5s 才能动轮
- 10 秒罚时(Stop&Go 10s):同上但 10s
- Drive-through:不停车开过 pit lane,损失 ~22 秒
- 赛后罚时:不停车,赛后从总用时减
- 网格 罚 5 / 10 / 网格末位 / 黑旗禁赛(极少)
触发原因:抢跑 / 不安全释放 / 超速 / 撞车失利 / 多次黄旗超车 等。

# F1 红黄旗 / 安全车

- 黄旗:某区段事故,该区禁止超车
- 双黄旗:严重事故,所有车减速
- 安全车 (SC):全场跟车,不许超车,直到清场
- 虚拟安全车 (VSC):全场必须维持目标 delta 时间,无 SC 实车出场
- 红旗:停赛,所有车回 pit,清场后从原位重发
红旗后允许换胎(2025 改了)且不算正常进站。

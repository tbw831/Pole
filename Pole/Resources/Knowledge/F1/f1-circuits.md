# F1 摩纳哥赛道 (Monaco / Circuit de Monaco)

地点:摩纳哥蒙特卡洛街道。长 3.337 km,F1 现役最短赛道。
特点:窄、紧、几乎无超车点。pole 决定 70% 比赛结果。
著名弯角:Sainte Devote (T1) / Casino / Mirabeau / Loews 发夹弯(F1 最慢) / Tunnel(隧道高速) / Nouvelle Chicane / Tabac / 游泳池 / La Rascasse / Anthony Noghes(终点弯)。
策略:几乎一停或不停。雨天剧本完全不同,容易混乱出冷门(2024 Leclerc 正是雨后赢)。
经典战:1996 雨战 Panis 力斗夺冠;2018 Ricciardo 引擎掉马力坚持守一; Senna 6 冠王。

# F1 银石赛道 (Silverstone)

地点:英国北安普敦。长 5.891 km,18 弯。F1 发源地,1950 第一届世界锦标赛在此。
特点:高速 flow 弯组合 — Maggotts / Becketts / Chapel(连续高速 S 弯,F1 现役最快弯组之一)。
长直道 Hangar 配合 Stowe 重刹,适合 DRS 超车。
策略:通常一停,软中或软硬两停;天气善变是这赛道的灵魂。
经典战:Hamilton 8 次主场获胜(F1 史上最多本土胜利);1987 Mansell 神追 Piquet。

# F1 斯帕赛道 (Spa-Francorchamps)

地点:比利时阿登山林。长 7.004 km,F1 现役最长赛道,19 弯。
特点:Eau Rouge → Raidillon 上坡盲弯组合 = F1 史上最负盛名弯角,需全油门通过。
天气是赛道灵魂 — 阿登常年潮湿,7 公里跨多个微气候,常常一头干一头湿。
长直道 Kemmel 适合 DRS 超车。Pouhon 和 Blanchimont 也是高速考验。
经典战:1998 Schumacher 雨中追上 Hill;2008 Hamilton 在最后两圈雨中超 Raikkonen 反被罚去 P3;2021"半程"红旗(雨太大没跑就发奖金)。

# F1 蒙扎赛道 (Monza)

地点:意大利米兰北郊。长 5.793 km,1922 建成,F1 现役最老赛道之一。
特点:长直道 + 高速弯,空气阻力低,号称 "Temple of Speed"。
平均时速 F1 历史第二高(仅次于已停的 Reims)。
关键弯:Variante del Rettifilo(T1 chicane,常发生大乱);Curva Grande;
Lesmo 双弯;Ascari chicane;Parabolica(终点高速长弯)。
策略:几乎一停,软硬胎,重赛车设定要"低翼"。
经典战:Ferrari 主场,Tifosi 风暴。1971 Peterson / Cevert / Hailwood / Ganley 0.6 秒内 5 车冲线。

# F1 苏佐卡赛道 (Suzuka)

地点:日本三重县。长 5.807 km,18 弯,1962 本田建。
特点:F1 现役唯一 8 字交叉布局,高速技术弯密集。
著名弯角:130R(高速左弯,~310 km/h 左右过) / Spoon Curve(双发夹弯)/ Esses(连续高速 S 弯)。
策略:两停常见,中硬胎为主。雨季 6 月日本梅雨期常下,雨战经常出戏。
经典战:Senna vs Prost 1989/1990 撞车决冠;2005 Raikkonen 17 位发车赢冠;2024 Verstappen 雨中破纪录。
